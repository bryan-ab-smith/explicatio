About

explicatio is a small and simple program to help facilitate text analysis.

Install

Windows
First, you need to install Python. You can get it here. Once Python is installed, you need to open up a command prompt and execute the following:

python -m pip install https://bryanabsmith.com/explicatio/releases/explicatio-current.whl

macOS (and Linux)
If you are running on Linux or a Mac, you've already got Python installed so all you need to do is execute the following:

python3 -m pip install https://bryanabsmith.com/explicatio/releases/explicatio-current.whl

Usage

Using explicatio requires launching it. As a command line tool, you are required to launch it from the command line.

Windows
Open up PowerShell, available in the Windows PowerShell folder in the Start menu.

explicatio

macOS
Open up /Applications/Terminal.app and simply execute explicatio:

explicatio

Commands

Note: the command examples below are all based on an analysis of the included Dracula text.

File Commands
Load File
load
Description
Load a file for analysis. You can pass anything that is supported by textract. This is a flexible library that can import a lot of content but you are strongly encouraged to ensure that the file is loaded as you wish by executing showcontents.
Parameters
filename: This must be a resolvable path to the file. If in doubt, use an absolute file path.
Usage
load [filename]
Example
load dracula.txt
Output
None.

Show Contents
showcontents
Description
Show the contents of the file that explicatio is working with. This is helpful to ensure that explicatio is working with the text in a form that you accept for analysis. If you import a text, it is possible for parts to be excluded or converted in a way that challenges accurate analysis. This function gives you the chance to make sure that you and explicatio are working with the same text.
Parameters
None
Usage
showcontents
Example
showcontents
Output
What explicatio is working with once a file is loaded.

Analytic Commands
Sentiment Analysis
sentiment
Description
Run a sentiment analysis on the text. This can be used to give you a sense of the emotional tone of the text. The function returns four values, one of which consolidates the other three to give you an overall score for the sentiment expressed in the text.
Parameters
None
Usage
sentiment
Example
sentiment
Output
Four values:
  • Negative: the negative score.
  • Positive: the positive score.
  • Neutral: the neutral score.
  • Compound: the compound score (what you're likely looking for).

Dispersion Plot
dispersion
Description
This can be used to see where a list of words are in the text and compare them against each other. In effect, this shows you the spread of words across a text and in allowing you to do this with multiple words, you can see how the spread of various word compare.
Parameters
list_of_words: A semi-colon delimited set of words. See usage below for an example.
Usage
dispersion [list_of_words]
Example
dispersion dracula;vampire
Output

Concordance
concordance
Description
Run a concordance. This can be used to see where a word is sitauted in the text with the context around it. Running a concordance can thus provide insight into the contextual use of the word by providing a sense of the words before and after it.
Parameters
word: The word to run the concordance on.
num_results: Number of results to show. It is possible that the number of results shown will be lower than the value provided here if there are fewer results than the value requested.
Usage
concordance [word] [num_results]
Example
concordance dracula 10

Collocations
collocations
Description
Run a collocation. This can be used to see what words are commonly collocated (or co-located) with each other. This can help with finding common noun phrases for example.
Parameters
None.

collocations

freqdist
Description

freqdist [num_words]

freqdist 10
