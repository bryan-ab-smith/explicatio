About
explicatio is a small and simple program to help facilitate text analysis.

Install
Windows
First, you need to install Python. You can get it here. Once Python is installed, you need to open up a command prompt and execute the following:

python -m pip install https://bryanabsmith.com/explicatio/releases/explicatio-current.whl

macOS (and Linux)
If you are running on Linux or a Mac, you've already got Python installed so all you need to do is execute the following:

python3 -m pip install https://bryanabsmith.com/explicatio/releases/explicatio-current.whl

Usage
Using explicatio requires launching it. As a command line tool, you are required to launch it from the command line.
macOS
Open up /Applications/Terminal.app and simply execute explicatio:

explicatio
Windows
Open up PowerShell, available in the Windows PowerShell folder in the Start menu.

explicatio

Commands
Load File
load
Description
Load a file for analysis. You can pass anything that is supported by textract. This is a flexible library that can import a lot of content but you are strongly encouraged to ensure that the file is loaded as you wish by executing showcontents.
Parameters
filename: This must be a resolvable path to the file. If in doubt, use an absolute file path.
Usage
load [filename]
Example
load dracula.txt
Output
None.

Dispersion Plot
dispersion
Description
This can be used to see where a list of words are in the text and compare them against each other. In effect, this shows you the spread of words across a text and in allowing you to do this with multiple words, you can see how the spread of various word compare.
Parameters
list_of_words: A semi-colon delimited set of words. See usage below for an example.
Usage
dispersion [list_of_words]
Example
dispersion dracula;vampire
Output
explicatio
