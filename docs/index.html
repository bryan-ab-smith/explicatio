About
explicatio is a small and simple program to help facilitate text analysis.

Install
Before you start, you need to install Python. If you are running on Linux or a Mac, you've already got it installed. If you're on Windows, you will need to download it. You can get it here.

Once Python is installed, you can execute the following:

macOS and *nix
python3 -m pip install https://bryanabsmith.com/explicatio/releases/explicatio-current.whl
Windows
python -m pip install https://bryanabsmith.com/explicatio/releases/explicatio-current.whl
